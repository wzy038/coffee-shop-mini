# UI界面截图

本目录存放小程序和管理后台的界面截图。

## 小程序页面

1. **首页** - 轮播图、导航菜单、热门商品
2. **商品列表页** - 分类筛选、商品展示
3. **商品详情页** - 商品信息、加入购物车
4. **购物车页** - 商品列表、数量调整
5. **结算页** - 订单确认、地址选择
6. **订单列表页** - 订单状态、订单详情
7. **AI推荐页** - 聊天对话、推荐卡片

## 管理后台页面

1. **仪表盘** - 数据统计、快捷入口
2. **商品管理** - 商品列表、新增/编辑商品
3. **订单管理** - 订单列表、状态管理
4. **用户管理** - 用户列表
5. **分类管理** - 分类列表、新增/编辑分类

## 命名规范

`页面名称.png`

示例：
- home.png
- products.png
- cart.png
- admin-dashboard.png
