# AI对话截图

本目录存放AI饮品推荐功能的对话截图。

## 测试场景

1. **减肥/健身场景**
   - 用户输入："我最近在减肥，推荐点适合的饮品"
   - AI推荐：低热量、无糖饮品

2. **熬夜/提神场景**
   - 用户输入："熬夜加班，推荐提神的饮品"
   - AI推荐：高咖啡因、浓郁饮品

3. **约会场景**
   - 用户输入："约会喝什么比较好"
   - AI推荐：颜值高、口感好的饮品

4. **日常场景**
   - 用户输入："推荐一款经典咖啡"
   - AI推荐：经典款饮品

5. **搭配推荐**
   - 用户输入："我买了拿铁，还想加点什么"
   - AI推荐：搭配的甜点或其他饮品
