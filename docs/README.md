# 项目文档目录

本目录存放项目相关的截图、演示录屏和接口测试文档。

## 文件结构

```
docs/
├── screenshots/          # 项目截图
│   ├── api-test/        # 接口测试截图
│   ├── ui/              # UI界面截图
│   └── ai-conversation/ # AI对话截图
├── demo/                # 项目演示录屏
└── API测试文档.md        # 接口测试说明文档
```

## 截图说明

### AI对话截图
- 展示AI饮品推荐功能的实际对话效果
- 包含用户输入和AI回复的完整流程

### 接口测试截图
- 使用Postman或其他工具测试API接口
- 包含请求参数和响应结果

### UI界面截图
- 小程序各页面的界面展示
- 管理后台的界面展示

## 使用说明

1. 将截图文件放入对应目录
2. 截图命名格式：`功能名称_日期.png`
3. 演示录屏使用MP4格式
